# 8조 리테일 – 한 페이지 요약

## 문제

- 4,324명 고객에 한정된 마케팅 예산으로 **누구에 집중할지** 결정해야 함.
- 고객 가치가 불균형한지(파레토) 검증하고, 세그먼트별 차별화 전략이 필요함.

## 접근

1. **전처리**: 취소·반품·무효 StockCode 제거 → `전처리_완료.csv`
2. **RFM**: R(Recency), F(Frequency)만으로 4세그먼트 분류. M(Monetary)은 결과 변수이므로 분류 기준에서 제외(순환 논리 방지).
3. **통계 검증**: Monetary 로그 변환 후 ANOVA·Kruskal-Wallis로 세그먼트 간 차이 유의성 검정.
4. **시각화**: 파레토 차트, 세그먼트별 도넛, ARPU·시간대별 히트맵 등.

## 결과

- **파레토**: 상위 20% 고객 = 매출의 **73.5%**
- **상위 20% 내부**: 이탈위험+충성 = 고객 91.7%, 매출 95.7% → 이 두 그룹에 마케팅 집중 권장
- **이탈위험**: 10–11월 매출 **-84.2%** 감소 → 재구매 유도 캠페인 필요
- **검정**: ANOVA·Kruskal-Wallis p<0.001로 세그먼트 구분 통계적으로 유의
